# Week Trainer

A phone-first workout app — runs and logs a weekly gym + cardio program.

- Works offline once added to your home screen (Add to Home Screen).
- Your data stays on your phone (no account, no server).
- Back up or move to a new phone any time via **⚙ Tools → Export / Import** (JSON).

Just a static web app: plain HTML/CSS/JS + IndexedDB, no build step, no dependencies.
