# Week Trainer — Exercise Pool (Tanner's gym / Bodifi)

Master list of available exercises, grouped by the muscle they primarily train, built from
the confirmed equipment set (2026-06-19). Each entry: **exercise — equipment — primary / secondary muscles**.
Muscle keys match the app (`js/muscles.js`) so this drops straight into the rotating-plan build.

This is the POOL the "set days, rotating exercises" system will draw from — nothing is locked to a day.
PERSONAL-ONLY cautions are tagged ⚠ (scrub these from the public/friends build).

- Cable crossover (high/mid/low) — cable crossover — chest
